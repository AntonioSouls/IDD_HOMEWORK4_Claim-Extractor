The user wants to extract claims from a given JSON object containing a table, caption, and references.
Each claim should be a JSON object with the following structure:
```json
{
    "0": {
        "specifications": {
            "0": {"name": "column_name_1", "value": "row_value_1"},
            "1": {"name": "column_name_2", "value": "row_value_2"}
        },
        "Measure": "measure_from_table_caption_or_references",
        "Outcome": "value_from_the_table_associated_to_the_measure"
    },
    "1": { ... }
}
```
The table can be of four types: relational, nested relational, cross table, and nested cross table.
